네이버 지도 Client ID 입력

페이지를 처음 사용하려면 네이버 클라우드 플랫폼에서 발급받은 Maps JavaScript API용 Client ID가 필요합니다. 입력한 값은 이 브라우저에만 저장돼요.

네이버 지도 인증에 실패했어요. 아래 자주 겪는 원인을 먼저 확인해주세요.
  • 키가 네이버 클라우드 플랫폼에서 발급된 키인지 확인 (네이버 개발자센터 키는 지도에 사용할 수 없어요)
  • NCP 콘솔의 Web 서비스 URL에 지금 이 페이지를 여는 주소가 등록돼 있는지 확인 (예: http://localhost:8000)
  • file:// 로 직접 더블클릭해 여는 경우 인증이 안 되는 경우가 많아요. 파일이 있는 폴더에서 python3 -m http.server 8000 으로 로컬 서버를 띄우고 http://localhost:8000/office-map.html 으로 접속해보세요
  • 아래 "발급 콘솔" 항목을 신규/구 버전으로 바꿔서 다시 시도해보세요
  • Maps JS API는 Client Secret을 쓰지 않아요. Secret과 함께 발급된 키라면 검색/번역용 키일 가능성이 높습니다
  1. 네이버 클라우드 콘솔 → Application 접속
  2. Application 등록 후 Maps 서비스 추가
  3. Web 서비스 URL에 현재 페이지를 여는 주소를 등록 (예: http://localhost:8000)
  4. 발급된 Client ID 값을 아래에 붙여넣기
잘 모르겠으면 신규를 두고 저장 → 실패 시 구 버전으로 바꿔보세요. 자동으로도 한 번 폴백을 시도합니다.
